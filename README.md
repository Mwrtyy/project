# Shahine — Premium Video Editor Portfolio

Portfolio audiovisuel dark luxury créé en HTML, CSS et JavaScript vanilla.

## Fonctionnalités

- Compatible GitHub Pages
- Design sombre premium : chrome, liquid glass, grille subtile, particules
- iPhone 3D en CSS/HTML avec Dynamic Island, bords métalliques, boutons latéraux et reflet de verre
- Rotation 3D interactive au curseur
- Navigation entre sections : Home, Edits, Skills, Contact
- Swipe mobile dans l'écran du téléphone
- Barres de progression animées
- Responsive desktop et mobile
- Aucune dépendance externe

## Lancer en local

Ouvre directement `index.html` dans ton navigateur.

Option plus propre avec un mini serveur local :

```bash
python -m http.server 5500
```

Puis ouvre :

```text
http://localhost:5500
```

## Publier sur GitHub Pages

1. Va dans les paramètres du repository GitHub.
2. Ouvre **Pages**.
3. Dans **Build and deployment**, choisis **Deploy from a branch**.
4. Sélectionne la branche `main` et le dossier `/root`.
5. Clique sur **Save**.
6. GitHub donnera une URL publique du type : `https://mwrtyy.github.io/project/`.

## Personnalisation rapide

- Remplace les liens TikTok / Instagram / Email dans `index.html`.
- Modifie les statistiques dans les sections Home.
- Remplace les cartes fictives par de vraies miniatures ou vidéos dans la section Edits.
