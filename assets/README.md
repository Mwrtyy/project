# Assets

Ce dossier est prévu pour les médias du portfolio.

- `images/` : miniatures, covers, stills.
- `videos/` : extraits vidéo compressés.
- `textures/` : textures WebP/AVIF légères.
- `models/` : modèles GLB/GLTF optimisés si tu veux remplacer le téléphone procédural.

Le téléphone actuel est généré en Three.js, donc aucun modèle 3D lourd n'est requis.
